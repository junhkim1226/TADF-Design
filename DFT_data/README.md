# Dataset Description

This directory contains CSV files with the raw data used and generated in this study.

## File Descriptions

- **`training_data.csv`**
  - Contains the training dataset used for model training.

- **`generated_blue_candidates.csv`**
  - Contains the generated candidate molecules targeting Blue TADF emission.

- **`generated_green_candidates.csv`**
  - Contains the generated candidate molecules targeting Green TADF emission.

- **`generated_red_candidates.csv`**
  - Contains the generated candidate molecules targeting Red TADF emission.

- **`random_enumerated_candidates.csv`**
  - Contains molecules generated through random enumeration for comparison.
